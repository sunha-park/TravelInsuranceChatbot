# 🛫 해외여행보험 챗봇 _(개인)_

> 2024.10.21 ~10.29 (10일)
> 
> **[주요기능]** <br />

> ② **LLM, RAG** 시스템 챗봇 서비스 <br />
>
> **[FrontEnd]**  HTML, CSS <br />
>
> **[BackEnd]**  Python, Flask, GCP, MariaDB
